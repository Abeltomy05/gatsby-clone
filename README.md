## **Gatsby Website Clone (HTML • CSS • JavaScript)**
A fully responsive clone of the Gatsby website homepage, built using HTML, CSS, and JavaScript without any external frameworks.

**Features**
- Pixel-Perfect UI — Faithfully recreates the Gatsby website design.
- Fully Responsive — Works seamlessly across desktops, tablets, and mobile devices.
- Clean & Organized Codebase — Semantic HTML, modular CSS, and readable JS.

**Tech Stack**
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6)

**Setup & Usage**
Clone the repository:
```bash
git clone https://github.com/Abeltomy05/gatsby-clone.git
```
Navigate into the project:
```bash
cd gatsby-clone
```
Open the website in your browser:
Simply open:
```bash
index.html
```
